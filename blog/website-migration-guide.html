Әзірлеу

Сайтты жаңа хостингке көшіру: қадамдық нұсқаулық

Жандос Арлан
22 қараша 2024 · 11 мин оқу

Сайтты деректерді жоғалтпай және тоқтаусыз қалай қауіпсіз тасымалдау керек. Бэкаптардан тестілеуге дейін.

1. Миграцияға дайындық

  • Толық бэкап жасау
  • Жаңа хостингті тексеру
  • DNS TTL төмендету

2. Файлдарды тасымалдау

FTP немесе SSH арқылы барлық файлдарды көшіріңіз.

3. Деректер қорын тасымалдау

phpMyAdmin арқылы экспорт/импорт жасаңыз.

4. Конфигурацияны жаңарту

wp-config.php файлындағы деректер қоры параметрлерін жаңартыңыз.

5. DNS жаңарту

Домен DNS жазбаларын жаңа серверге бағыттаңыз.

6. Тестілеу

  • Барлық беттерді тексеру
  • Формаларды тестілеу
  • SSL жұмысын тексеру

Қорытынды

Миграция қиын емес, бірақ мұқият жоспарлауды қажет етеді. Әрқашан бэкап жасаңыз!
