WordPress деректер қорын оңтайландыру: сайтты 40%-ға жылдамдату

Нұрхатова Райхан
25 қараша 2024 · 8 мин оқу

WordPress ДҚ тазалау және оңтайландыру техникалары. Максималды өнімділік үшін плагиндер мен қолмен әдістер.

1. Неге ДҚ оңтайландыру маңызды?

Уақыт өте келе WordPress деректер қоры қоқыспен толады:

  • Пост ревизиялары
  • Спам пікірлер
  • Transient опциялары
  • Жойылған плагиндер деректері

2. Қолмен тазалау

Ревизияларды жою:

DELETE FROM wp_posts WHERE post_type = 'revision';

Спам пікірлерді жою:

DELETE FROM wp_comments WHERE comment_approved = 'spam';

3. Плагиндер арқылы оңтайландыру

  • WP-Optimize: Кешенді тазалау
  • Advanced Database Cleaner: Терең тазалау
  • WP-DBManager: ДҚ басқару

4. Кестелерді оңтайландыру

phpMyAdmin арқылы немесе SQL сұрауымен:

OPTIMIZE TABLE wp_posts, wp_postmeta, wp_options;

5. Ревизияларды шектеу

wp-config.php файлына қосыңыз:

define('WP_POST_REVISIONS', 3);

6. Автоматты тазалау

WP-Optimize плагинінде кесте орнатыңыз.
